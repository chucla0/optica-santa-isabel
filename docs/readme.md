##Stack Tecnológico

### Core
- **Next.js 15 (App Router)**: Framework principal para React.
- **TypeScript**: Para un desarrollo robusto y tipado.
- **Tailwind CSS**: Motor de diseño para estilos modernos y responsivos.

### Librerías de Interacción
- **Framer Motion**: Para transiciones de página y animaciones de scroll.
- **Lenis**: Implementación de scroll suave (smooth scroll).
- **Rive**: Integración de animaciones vectoriales colaborativas (como el ojo interactivo).

### Backend & Servicios
- **NodeMailer**: Gestión de envíos de correo vía SMTP.
- **Vercel**: Plataforma de hosting y despliegue continuo.
